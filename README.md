## Описание проекта
В данном репозитории реализованы две задачи на чистом JavaScript (без использования jQuery):

### 1. Виджет Popover
Аналог виджета всплывающих подсказок из Bootstrap.
* Появляется по клику на кнопку.
* Позиционируется строго над элементом (в пикселях, без использования translate).
* Центрируется по горизонтали относительно элемента, который его вызвал.

### 2. Редактор списка (CRM)
Виджет для управления списком товаров с поддержкой базовых CRUD-операций:
* **Create & Update:** Добавление новых и редактирование существующих записей через всплывающее модальное окно. Форма корректно сбрасывает состояние при открытии после прошлых действий.
* **Read:** Отображение товаров в виде таблицы
* **Delete:** Удаление товаров с предварительным всплывающим окном подтверждения действия.
* **Валидация:** Проверка данных перед сохранением (название не может быть пустым, стоимость должна быть числом больше нуля). Ошибки отображаются текстом под соответствующими полями.
* **Хранение:** Все данные хранятся в памяти (в виде JS-объектов), а DOM-дерево перестраивается на основе этих данных, реализвано как на макетах точь в точь

## Технологии
* **JS:** Vanilla JavaScript (ES6+)
* **Сборщик:** Webpack (с настроенными loader'ами для CSS и HTML), Babel
* **Тестирование:** Jest, Puppeteer (E2E-тесты на проверку отображения Popover и валидации формы CRM)
* **CI/CD:** GitHub Actions (автоматический линтинг, прогон тестов и деплой на GitHub Pages)
